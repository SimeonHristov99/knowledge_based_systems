Task 2:
    Topic: ???
    Definition for a semantic net: ???
    Definition for a partitioned semantic net: ???
    When do we use semantic nets?
        When sentence does not contains quantifiers (all, every, some, any),
        but only words like "a" and "the"
    When do we use partitioned semantic nets?
        When sentence contains quantifiers (all, every, some, any)

    Keywords:
        is-a
        agent: object of a positive action
        beneficiary: subject of a positive action
        assailant: object of a negative action
        victim: subject of a negative action
        SA (Scene Area): Holds all scenes
        Si (scene) (semantic net)
        form: points to a scene
        quantifier (every, all, some): points to the node(s) it is for

    Create semantic nets for the following sentences:

    1. The dog bit the mail carrier.
    2. Every dog has bitten a mail carrier.
    3. Every dog in town has bitten the constable.
    4. Every dog has bitten every mail carrier.
    5. Every player kicked a ball.

    (HW) All players liked the referee.
    (HW) Some dogs ate all bones.
    (HW) All Englishmen like the Queen.

    Don't forget: Only relational constants without "the"
    go into the partitioned semantic network!
